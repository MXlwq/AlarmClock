# AlarmClock
想要实现一个比普通闹钟更容易把人叫醒的闹钟。  
和普通的闹钟相比，主要想实现:  
1. 摇手机N次关闹钟、拍同一场景关闭闹钟、算算术题关闭闹钟  
2. 闹钟响起之后不能关机、不能减小音量，只有满足自己之前设定的条件才能停止
